
### Download & Install

Download the Docker CE for Mac App here: [https://download.docker.com/mac/stable/Docker.dmg](https://download.docker.com/mac/stable/Docker.dmg)

Follow the installation instructions for Mac here:
[https://docs.docker.com/docker-for-mac/install/#install-and-run-docker-for-mac](https://docs.docker.com/docker-for-mac/install/#install-and-run-docker-for-mac)


Once complete, open a terminal and run `docker info` and verify that you see output
